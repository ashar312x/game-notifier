# Game Notifier

Automatically sends a WhatsApp message to a group chat the moment you launch a game. Runs silently in the system tray and requires no manual effort once configured.

## How it works

1. The app polls the Windows process list every 5 seconds via `tasklist`.
2. When a tracked game executable appears, it waits a configurable delay (default 10 s) to confirm the game is actually running.
3. It then sends a message to your WhatsApp group by automating **WhatsApp Desktop** using PowerShell — no browser, no third-party API.
4. A cooldown (default 30 min) prevents repeated messages for the same game.

> **Requirement:** [WhatsApp Desktop](https://www.whatsapp.com/download) must be installed. The app will launch it automatically if it is not already open.

## Installation

```powershell
git clone <repo-url>
cd game-notifier
npm install
npm run setup
```

`npm run setup` generates `assets/icon.png` without any extra dependencies.

---

## Running

```powershell
npm start
```

The app starts minimised to the system tray. On the first launch, the Settings window opens automatically so you can enter your group name.

---

## Configuration

Double-click the tray icon (or right-click → **Settings…**) to open the settings panel.

| Setting | Description | Default |
|---------|-------------|---------|
| WhatsApp group name | Exact name of the group to message | `Game on` |
| Message template | Use `{game}` as a placeholder for the game name | `Hey I am playing {game}, do you guys wanna hop in? 🎮` |
| Cooldown | Minutes before the same game triggers another notification | `30` |
| Detection delay | Seconds a process must be running before a notification is sent | `10` |
| Enabled | Toggle notifications on/off from the tray menu | On |

Config is stored at `%APPDATA%\game-notifier\config.json`.

### Games list

The app ships with 19 popular games pre-configured (CS2, Valorant, Fortnite, GTA V, Apex Legends, and more). You can add, remove, or rename games in the Settings panel. Each entry needs:

- **Name** — displayed in the notification and the message
- **Executable** — the `.exe` filename as it appears in Task Manager (e.g. `cs2.exe`)

## Building a distributable

Install `electron-builder` and build a Windows installer:

```powershell
npm install --save-dev electron-builder
npx electron-builder --win --x64
```

The installer will be placed in the `dist/` folder. You can also add a build config to `package.json`:

```json
"build": {
  "appId": "com.yourname.game-notifier",
  "productName": "Game Notifier",
  "win": {
    "target": "nsis",
    "icon": "assets/icon.png"
  },
  "nsis": {
    "oneClick": true,
    "perMachine": false
  }
}
```

Then run:

```powershell
npx electron-builder --win
```

## Troubleshooting

**Message goes to the wrong chat** — Make sure the group name in Settings matches the WhatsApp group name exactly (case-sensitive).

**WhatsApp takes focus unexpectedly** — This is by design. The automation must bring WhatsApp to the foreground to interact with it. It restores WhatsApp to its previous minimised state after sending.

**"Could not find WhatsApp window"** — WhatsApp Desktop is not installed or failed to start within 25 seconds. Install it from [whatsapp.com/download](https://www.whatsapp.com/download).

**No notification for a game** — Check that the executable name in the games list matches exactly what appears in Task Manager (Windows key + X → Task Manager → Details tab).
